Configuration Telegram API

Configuration Telegram API

Definition

API stands for Application Programming Interface. The term "Application" in API refers to software with different functions.

How to Create a Telegram API

  • Please visit the link https://my.telegram.org/auth
  • Enter your Telegram number
  • Enter the confirmation code via Telegram (not SMS), click [Sign In]
  • Visit Telegram's official website and choose API development tools from the Your Telegram Core menu.
  • After selecting API development tools, you will be directed to https://my.telegram.org/apps. Create New Application ➔ fill in the provided fields.
FieldDescriptionExample
App titleThe title of your app (Feel free to name your app as you wish)Wili Sandi
Short nameYour short Telegram nameWili
URLSkip if you don't have a website-
PlatformChoose the platform your application is made forDesktop
DescriptionFill in as you wish-

Once you have completed the fields, click [Create Application] to save the new API you just created.
