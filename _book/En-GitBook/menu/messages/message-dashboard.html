Message Dashboard

Message Dashboard

All relevant information regarding the Order menu is provided concisely, including several informational graphs about agents.

In this display, there are several sections, each with its own function. The explanation of these functions is as follows:

Features/ButtonsFunction
UnassignedTotal of all conversations not yet assigned to agents
AssignedTotal of all conversations assigned to agents.
ResolvedSum of all conversations already answered, and the agent wants to end the conversation.
ArchivedTotal from all archived conversations
Created ConversationInformation on the graph of all received and sent conversations (per hour or per day)
Agent & SupervisorInformation on the company's total number of members.
Conversation TagsConversation graph information based on tags
Agent Online StatusAgent Graph Information Based on Online Status
Conversation FunnelConversation chart information based on the funnel.
